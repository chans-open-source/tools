# dubbogo-cli-v2
> dubbo-go 集成工具

## 使用方式
1. 安装
```bash
go get -u github.com/dubbogo/tools/cmd/dubbogo-cli-v2
```
## 主要功能

### 获取接口及方法列表
```bash
./dubbogo-cli-v2 show --r zookeeper --h 127.0.0.1:2181
```
